📝 How to Add Products with Images

# Using curl:
curl -X POST http://localhost:3000/api/livestock \
  -F "farmer_id=1" \
  -F "name=Sheep" \
  -F "type=Merino" \
  -F "quantity=10" \
  -F "price=5000" \
  -F "description=Healthy sheep for sale" \
  -F "image=@/path/to/image.jpg"
